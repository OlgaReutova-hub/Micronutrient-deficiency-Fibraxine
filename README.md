# Пример рациона со сниженной калорийностью

Веб-приложение на [Streamlit](https://streamlit.io): подбор типового меню на день, отображение дефицита пищевых волокон и железа, генерация HTML-памятки для пациента.

## Требования

- Python 3.10+ (в коде используются аннотации вида `dict | None`)

## Установка и запуск

```bash
cd "путь/к/папке/проекта"
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

Браузер откроется автоматически. Кнопка «Памятка для пациента» сохраняет файл `data/patient_memo.html` и открывает его в приложении по умолчанию (на Windows — через `os.startfile`).

## Структура проекта

| Файл / папка | Назначение |
|--------------|------------|
| `app.py` | Точка входа Streamlit |
| `requirements.txt` | Зависимости (Streamlit) |
| `data/menu 1200 - 2000.json` | Сценарии меню и дефициты по калорийности и типу питания |
| `data/Шаблон памятки.html` | Шаблон HTML с плейсхолдерами `{{...}}` для памятки |
| `data/patient_memo.html` | Создаётся при работе приложения (можно не коммитить) |

## Зависимости

В `requirements.txt` указан только **Streamlit** (версия ≥ 1.44.0). Остальные импорты в `app.py` — стандартная библиотека Python (`json`, `os`, `platform`, `subprocess`, `pathlib`).
